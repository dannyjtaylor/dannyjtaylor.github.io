How to Play — Voltbox Help

Welcome to Voltbox!

Build circuits on the virtual breadboard:

Red rails (+) = Power (5V)

Blue rails (−) = Ground (0V)

Each column of 5 holes (a–e, f–j) is connected internally.

Building a Circuit:

  1. Select Wire and click two holes to connect them
  2. Place a Resistor between two holes
  3. Place an LED (first click = anode +, second = cathode −)
  4. Or place Power / Ground markers on any hole

Controls:

  • Left click — Place component pins
  • Right click — Delete component
  • 1–7 — Quick-select tools
  • Space — Run simulation
  • Ctrl+Z — Undo

Try this: Power rail → wire → resistor → LED → wire → ground rail
